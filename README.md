# Employee Search Tool 
Version: 1.0.0

# Overview

The Employee Search Tool is a demonstration that illustrates how easily leaked sensitive employee information can be misused. The 2023 Amazon employee database breach exposed the contact details of 1.1 million employees, which are **still active** and easily accessible online thorugh various forums. This creates a high risk for social engineering attacks. 

# ⚠️ Important:

## This tool does not include the breached data or directly links to it.
## It simulates searching for employee information for educational purposes only. Based on excel file format or parquat.
