# Employee Search Tool (Demonstration Only)
Version: 1.0.0

# Overview

The Employee Search Tool is a demonstration that illustrates how easily sensitive employee information can be exposed and misused. The 2023 Amazon employee database breach exposed the contact details of 1.1 million employees, which are still active and easily accessible online. This creates a high risk for social engineering attacks.

# ⚠️ Important:

This tool does not access or link to any real data.
It simulates searching for employee information for educational purposes only.
I do not condone or engage in any illegal activities, and I am not a threat actor. The intent of this tool is to raise awareness of the potential for social engineering if exposed data is not properly secured.
